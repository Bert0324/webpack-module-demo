# webpack-module-demo

A simple Demo to show webpack build modules.

## Install

`yarn`
`yarn build`

If you want a module bundle run in the browser:

`MODULE=asyncImport yarn dev`

MODULE includes: 'es6', 'commonJS', 'libraryImport', 'asyncImport'.
